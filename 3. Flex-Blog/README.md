# Introdução
O FlexBlog foi construído a partir do curso CSS Flexbox oferecido pela Origamid, instruído por André Rafael.

### Neste projeto foram utilizados os seguintes conceitos de Flexbox:

- Display Flex
- Flex Direction
- Flex Wrap
- Flex Flow
- Justify Content
- Align Items
- Align Content
- Flex Grow
- Flex Basis
- Flex Shrink
- Flex
- Order
- Align Self
